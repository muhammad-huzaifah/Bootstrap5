Vertical Direction

Use .flex-column to display the flex items vertically (on top of each other).

Use .flex-column-reverse to reverse the vertical direction, and use .flex-sm|md|lg|xl-column-reverse to achieve this on different screen sizes.

.flex-column-reverse:

Flex item 1
Flex item 2
Flex item 3

.flex-sm-column-reverse:

Flex item 1
Flex item 2
Flex item 3

.flex-md-column-reverse:

Flex item 1
Flex item 2
Flex item 3

.flex-lg-column-reverse:

Flex item 1
Flex item 2
Flex item 3

.flex-xl-column-reverse:

Flex item 1
Flex item 2
Flex item 3

.flex-xxl-column-reverse:

Flex item 1
Flex item 2
Flex item 3
